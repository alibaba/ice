`build-plugin-component` 提供组件开发时的基础能力扩展，本文将详细介绍插件具体用法。

## 功能

- 支持组件开发 demo 展示
- 支持组件 es 和 lib 目录的构建

## 如何使用

```bash
$ npm i --save-dev build-plugin-component
```

Usage:

```json
{
  "plugins": [
    "build-plugin-component"
  ]
}
```

更多组件开发相关内容，详见[业务组件开发规范](/docs/materials/react.md)。
