> 在修改文档之前，请先阅读此文章

### 目录结构

```
- foo.md      文档内容, 可以有文件夹嵌套
- yoo.md
```

### 文档结构

```markdown
---
title: 文档标题(必须有)
order: 可选, 文档顺序, 数字越小越在前面, 否则按照字母序
hide: 可选, 布尔值, 是否隐藏文档

cover: 可选, 建议有, 封面图
---

markdown 格式的文档内容
```

### 链接跳转

统一使用绝对路径跳转，比如：`[设计工具](/docs/design/tools.md)`

### 词汇表述

请注意大小写和复数形式

* `ICE`: 品牌名, 中文称之为 `飞冰`
* `iceworks`: 配套 GUI 软件
* `物料 - marterial`: 包含区块, 布局, 模板和组件
* `区块 - block`: 复用的最小代码片段
* `布局 - layout`: 为 ICE 提供整体布局方案的代码
* `模板 - scaffold`: 整站示例, 脚手架
