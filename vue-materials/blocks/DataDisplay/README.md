# data-display

简介：数据展示

用于简单数据的模板展示

![截图](https://img.alicdn.com/tfs/TB1IKT9jfDH8KJjy1XcXXcpdXXa-1892-324.png)

数据格式:

~~~
export default {
  span: 8,
  color: '#15A0FF',
  data: [{
      count: 100,
      title: '日活跃数',
    },
    {
      count: '3,000',
      title: '月活跃数',
    },
    {
      count: '20,000',
      title: '年活跃数',
    }
  ]
}
~~~
