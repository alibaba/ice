标题组件。

## 参数（props）

| 参数名     | 说明                             | 必填 | 类型                      | 默认值 | 可选值     | 备注 |
| ---------- | -------------------------------- | ---- | ------------------------- | ------ | ---------- | ---- |
| text       | 要显示的文本                     | 否   | string                    |        |            |      |
| subtitle   | 二级标题                         | 否   | string 或 React.component |        |            |      |
| decoration | 是否有左边竖线                   | 否   | boolean                   | true   | true/false |      |
| children   | 要显示的内容，支持复杂 component | 否   | string、components        |        |            |      |
