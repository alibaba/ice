## 参数（props）

| 参数名    | 说明           | 必填  | 类型             | 默认值 | 备注 |
| --------- | -------------- | ----- | ---------------- | ------ | ---- |
| className | 样式名         | false | string           | -      |      |
| style     | 样式           | false | object           | {}     |      |
| loading   | 加载时 loading | 否    | boolean          | false  |      |
| error     | 数据错误       | 否    | 任何数据类型的值 | false  |      |
| empty     | 数据为空       | 否    | 任何数据类型的值 | false  |      |
| title     | 标题           | 否    | string           | -      |      |  |
