富文本组件 html 渲染器

## 安装和升级

```bash
npm install @icedesign/richtext-renderer
```

### 何时使用

- 结合ICELAND富文本组件使用，用来渲染富文本输出的HTML字符串

## 参数（Props）

|  参数名  |    说明    |   必填    |    类型   |  默认值  |  可选值  |
|-------|-----------|----------|---------|---------| -------- |
| html | 富文本字符串 | 是 | `string` | 无 |  |
| className | class名 | 否 | `string` | 无 |  |
