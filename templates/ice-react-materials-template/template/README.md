# 自定义物料初始模板

## 简介

用于自定义物料，支持自定义区块、布局、模板

## 使用

#### 安装开发者工具

```
$ npm install ice-devtools -g
```

#### 区块预览

```
$ cd blocks/Gretting
$ npm install
$ npm run start
```

#### 模板预览

```
$ cd scaffolds/lite
$ npm install
$ npm run start
```

## 添加物料

```
# 根据提示自定义添加
# 需要到根目录下添加
$ ice-devtools add
```
