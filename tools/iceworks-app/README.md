# Iceworks App

## Develop

0. Start iceworks server: 

    ```bash
    $ cd ../../pacakges/iceworks-server
    $ npm run dev
    ```
0. Start iceworks client:

    ```bash
    $ cd ../../packages/iceworks-client
    $ npm run start
    ```
0. Start icewroks app:

    ```
    $ npm run start
    ```

## Build

```
$ npm run dist
```
