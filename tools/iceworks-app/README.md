# Iceworks App

## Develop

0. Start iceworks server: 
    ```bash
    $ cd ../../pacakges/iceworks-server
    $ npm install
    $ npm run build
    $ cd ../../tools/iceworks-app
    $ ln -s ../../packages/iceworks-server server    
    ```
0. Start icewroks app:
    ```bash
    $ npm run start
    ```

## Build

```
$ npm run dist
```
